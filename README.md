# 點燃科技 - 前端工程師面試

本專案使用 `create-react-app` 所建立，僅供 **點燃科技** 所使用。

### 安裝

專案現保存於本人 Github Private。
```sh
git clone https://github.com/explooosion/QBS-F2E-Interview.git
```

```sh
cd QBS-F2E-Interview
```

```sh
yarn install
```

### 啟動

```sh
yarn start
```

### 建置

```sh
yarn build
```

### 樣式風格檢測

```sh
yarn lint
```
